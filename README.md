# ufc-fight-predictor
Machine Learning model for predicting UFC fights.

Data scraping done by WarrierRajeev/UFC-Predictions.
Data preprocessing also heavily borrowed from WarrierRajeev/UFC-Predictions.

Ideas I would like to explore in the future:
* Addressing multicollinearity and examining it's effect on model performance.
* Investigating variable influence in XGBoost.
* The UFC has had distinct eras where fighting style has changed. What effect would removing fights from before 2001 (Zuffa era) have on the model?
* Could preprocessing be done differently, and more neatly?
